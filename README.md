# What is Twitch Information?

Have you ever wanted basic details on a Twitch Streamer? Are you a Developer who needs a Twitch User ID?

Go [here](https://streamerinfo.vercel.app/) to see the live version of the app.

### Build Setup

```bash
# install dependencies
$ npm install

# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start

# generate static project
$ npm run generate
```
For detailed explanation on how things work, check out [Nuxt.js docs](https://nuxtjs.org).
